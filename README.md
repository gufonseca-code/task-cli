# Task-CLI
## Gerenciador de tarefas para o terminal
Este programa foi feito em Python e as tarefas são armazenadas em JSON localmente. Nenhuma dependência é necessária para utilizá-lo.
### Comandos
- `add` adiciona uma tarefa.
- `update` atualiza uma tarefa a partir de um id.
- `delete` deleta uma tarefa a partir de um id.
- `mark-in-progress` marca uma tarefa como 'em progresso' a partir de um id.
- `mark-done` marca uma tarefa como 'feita' a partir de um id.
- `list` lista todas suas tarefas.
